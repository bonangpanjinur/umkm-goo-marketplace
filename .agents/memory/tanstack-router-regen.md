## Rule
When creating new route files in `artifacts/kopihub/src/routes/`, the running KopiHub Dev workflow auto-regenerates `routeTree.gen.ts` within seconds. Manual line-number-based patches to routeTree.gen.ts will be overwritten or interleave with the auto-generated content.

**Why:** TanStack Router Vite plugin watches for file changes and rebuilds the route tree automatically. The plugin inserts correct imports, constants, type maps, union types, RouteChildren interface/object, and path config objects.

**How to apply:** Just create the `.tsx` route file with correct `createFileRoute('/path')` call and the correct parent route — routeTree updates itself. Only manually edit routeTree if the dev server is NOT running (cold-start/CI).

**Caution:** If you run a Python line-number script while the dev server is running, line numbers shift underneath you. The dev server will regenerate after your script runs, so duplicates or misplacements may appear. Check with grep -c after any manual patch.
